# O Coração

Um site romântico feito pra ser personalizado e presenteado. Você coloca as fotos, as músicas, as cartas e os lugares. A pessoa que receber só precisa clicar no coração pra entrar num espaço inteiro dedicado a ela.

Clone, personalize, e manda o link.

## Como rodar

Clone o repositório e abra o `index.html` com o Live Server.

## O que tem dentro

**Oráculo**: frases carinhosas sortidas a cada visita

**Carta**: cartas de amor sortidas, editáveis no código

**Memórias**: galeria de fotos em estilo polaroid com legenda

**Sinfonias**: músicas do Spotify que marcaram o relacionamento

**Mapa**: fotos dos lugares especiais que vocês foram juntos

## Como personalizar

Tudo fica no `js/script.js`. Substitua as fotos na pasta `recursos/nos/` e `recursos/lugares/`, edite as cartas, as músicas e as frases do oráculo com a sua história.

## Stack

HTML, CSS e JavaScript puro. Sem frameworks, sem dependências externas.
